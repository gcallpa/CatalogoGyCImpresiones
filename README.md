# 🚀 gycimpresiones - Catálogo 3D Dinámico

SPA (Single Page Application) moderna, ligera y dinámica diseñada para exhibir el portafolio de productos de impresión 3D de **gycimpresiones**. 

## 🛠️ Stack Tecnológico

*   **Frontend:** React 18 + Vite (rápido y optimizado).
*   **Estilos:** Tailwind CSS v3 (diseño responsivo y limpio con colores sólidos).
*   **Animaciones:** Framer Motion (transiciones fluidas y feedback visual interactivo).
*   **Interactividad 3D:** Three.js / React Three Fiber (visualizador interactivo de filamentos en tiempo real).
*   **CDN de Imágenes:** Cloudinary (optimización automática y entrega rápida de assets).
*   **Despliegue & Hosting:** Firebase Hosting (despliegue seguro y de alta disponibilidad).

## 📦 Estructura del Proyecto

*   `/src/components`: Componentes reutilizables (tarjetas de productos, buscador, modal del configurador 3D).
*   `/src/data`: Base de datos local (JSON) con el mapeo de rutas optimizadas de Cloudinary.
*   `firebase.json` & `.firebaserc`: Configuraciones listas para desplegar directo a la infraestructura de Firebase.
